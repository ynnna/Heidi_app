Extinction

These simulations accompany the ones summarised in Honey, Dwyer and Iliescu (2020a; pp 838-839; Figure 5). The design of the simulated data consists of two phases, a training phase, where a CS is continuously reinforced (CS+, first half of the trials), followed by an extinction phase, where the CS is no longer reinforced (CS-, second half of the trials). The simulations illustrate HeiDI's prediction that CS-oriented behaviour (e.g., lever presses) extinguishes slower than US-oriented behaviour (e.g., food-well entries; see Iliescu, Hall, Wilkinson, Dwyer & Honey, (2018).

Panel A depicts simulations of the development of the CS-US associations, derived from Equation 1 (VCS-US), the US-CS associations, derived from Equation 2 (VUS-CS), and their combined values, generated from Equation 3 (VCOMB). During extinction, VCS-US and VCOMB decline, however VUS-CS remains unchanged.

The simulations shown in Panel B reflects the way RCS extinguishes slower than RUS.

Functionality

Simulations tab

HeiDI's predictions can be explored using the three sliders.

Sliders

  • Number of trials (range 1 - 100) - Sets the number of trials desired. The slider only allows even numbers, as the simulations assume training and extinction phases are equal in the number of trials.
  • α (range 0.1 - 1) - Sets the perceived value for the CS.
  • β (range 0.1 - 1) - Sets the perceived value for the US.

Buttons

  • button - Link to code of a standalone version, in R, on github.
  • button - downloads the csv file generated in the simulation. It corresponds to the same data that be visualised in the Data tab

Data tab

Every time sliders are adjusted, the simulation dataframe is re-created. This tab shows the data plotted in the figures.
