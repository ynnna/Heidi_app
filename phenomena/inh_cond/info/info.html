Inhibitory conditioning

These simulations accompany the ones sumarrised in Honey, Dwyer and Iliescu (2020a; pp 839-840; Figure 6). The design of the simulated data consists of the presentation of two type of stimuli: A (continous reinforced; A+) and a compound stimulus AB (non-reinforced; AB-). These trials are presented in a random pre-set manner (fixed seed). The simulations illustrate HeiDI's predictions for the combined strenght of these associations (VCOMB-A, VCOMB-AB) and performance (A RCS, AB RCS, A RUS and AB RUS).

Panel A depicts the development of the combined values of associative strenght derived from Equations 1-3, for each stimulus (VCOMB-A, VCOMB-AB). As training progresses, VCOMB-A for stimulus A increases, VCOMB-AB declines during training, reflecting the fact that B has acquired negative associative strength that counteracts the positive associative strength acquired by A increases.

The simulations shown in Panel B provide HeiDI's predictions for performance (A RCS, AB RCS, A RUS and AB RUS) derived from Equations 1-5. When α A and α B are set lower than βUS the difference between A and AB is greater for RUS than RCS, but when α A and α B are set higher than βUS the difference between A and AB is greater for RCS than RUS. This implies that the inhibitory properties of stimulus B differ depending on whether the excitor with which it was trained (A) primarily evoked CS-oriented (RCS > RUS) or US-oriented (RUS > RCS) behaviour. As far as we are aware, there have been no experimental investigations of this issue. HeiDI clearly predicts that individual differences in the tendency for excitatory CS or US-oriented behaviour will be reflected in the tendency for expression of inhibition. This novel prediction has yet to be investigated.

Simulations show changes across both type of trials (A+ and AB-), the shape of the plotted values indicating the type of trial.

Functionality

Simulations tab

HeiDI's predictions can be explored using the three sliders.

Sliders

  • Number of trials (range 1 - 100) - Sets the number of trials desired. The slider only allows even numbers, as the simulations assume equal number of A+ and AB- trials.
  • α A (range 0.1 - 1) - Sets the perceived CS value for stimulus A.
  • α B (range 0.1 - 1) - Sets the perceived CS value for the stimulus B.
  • β (range 0.1 - 1) - Sets the perceived value for the US.

Buttons

  • button - Link to code of a standalone version, in R, on github.
  • button - downloads the csv file generated in the simulation. It corresponds to the same data that be visualised in the Data tab

Data tab

Every time sliders are adjusted, the simulation dataframe is re-created. This tab shows the data plotted in the figures.
